# AWS
O que é a AWS:
* A AWS e´uma plataforma de nuvem segura que ofere um amplo conjunto de produtos globais baseados na nuvem.
 A AWS oferece acesso sob demanda a recursos de computação,armazenamento,rede,banco de dados e outros recursos de TI e ferramenta de gereciamento.
 A AWS oferece flexibilidade. Voc~e paga apenas pelos serviços individuais de que precisa,pelo tempo que os utilizar.OS serviços da AWS funcionam juntos como componentes básicos.

Infraestrutura Global:
* foi um projeto e criada para entregar um ambiente de computação em nuvem flexível,confiável,dimensionável, e seguro comalta qualidade global em desempenho de rede.
* Regiões e Zonas de Disponibilidade:
América do norte,América do sul,Europa, Oriental Médio, África, Ásia-Pacifico e Austrália.

Pagamentos:
*Apenas pelo que usar 
*pague menos ao fazer uma reserva
*pague menos quanto mais usar e a plataforma cresce

Tipos de Reserva:
AURI(ALL)
*instância reservada com pagamento anteciptado integral(MAIOR DESCONTO)
PURI(PACIAL)
*instância reservadas com pagamento antecipado parcial(DESCONTOS MENORES)
NURI(NULL)
*instância reservada sem pagamento antecipado(DESCONTO MENOR)
