 Data centers da AWS
 •Os data centers da AWS são projetadospara segurança.•Os data centers são onde os dadosresidem e o processamento dedados ocorre.

 •Cada data center tem energia,redeseconectividade redundantes eestá hospedado emuma instalação separada.•Normalmente, um data center tem de50.000 a 80.000 servidores físicos.
